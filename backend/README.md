# Pasos para poder hacer uso de la aplicacion

## Previo a correr el servidor se tiene que asegurar que se cuente con lo siguiente instalado
- Express
- cloudinary
- stripe

Esto se realiza con npm install desde la terminal del back.

## Correr el servidor
Para correr el servidor es necesario desde la terminal del back utilizar el comando.

**npm start**
